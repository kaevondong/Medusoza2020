# PoleRequest V1.0
Web Application for tracking utility poles.
Team Medusozoa: Kevin Dong, Andrew Lubinger, Gary Isufi, Noah Limes

## Tools Used
* Ruby 2.7.2+
* Rails 6.0.3.4+
* Yarn 1.22.5+
* npm 6.14.6+
* Nodejs 10.15.3+
* Boostrap 4.5.0+
* JQuery 3.5.1+
* popperjs 1.16.1+
* Apple Maps API

## Setup
1. Install Dependencies


        $ gem install bundler
        $ bundle install
        $ npm install


2. Run Local Server


        $ rails s


3. :beers: Enjoy :beers:
