# PoleRequest
Web Application for tracking utility poles.

## Dependencies
* Ruby Version 2.7.2
* Rails Version 6.0.3.4
* Boostrap 4.5.0
* Jquery
* Apple Maps API

## Setup
1. Install Dependencies


        $ gem install bundler
        $ bundle install


2. Run Local Server


        $ rails s


3. :beers: Enjoy :beers:
